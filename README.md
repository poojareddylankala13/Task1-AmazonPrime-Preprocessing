Task 1 - Amazon Prime Dataset Preprocessing

Objective
To clean and preprocess Amazon Prime dataset for machine learning.

Tools Used
- Google Colab
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

Steps Performed
1. Imported dataset
2. Checked missing values
3. Handled null values
4. Encoded categorical features
5. Standardized numerical columns
6. Visualized outliers using boxplot
7. Removed outliers using IQR
8. Generated correlation heatmap
9. Saved cleaned dataset

Dataset
Amazon Prime Movies and TV Shows Dataset from Kaggle
